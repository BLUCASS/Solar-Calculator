# Bright Future Calculator

A modern, production-style solar quote calculator built with pure HTML, CSS, and vanilla JavaScript for fast internal quoting by solar sales teams.

## Overview

Bright Future Calculator helps quickly generate accurate solar quotes with live pricing for:

- Solar panel packages.
- Battery storage.
- Ground mount systems.
- Optimizers.
- Additional add-ons.
- Automatic BER certificate inclusion.

The interface is designed to feel like a premium SaaS dashboard, with a clean layout, responsive design, dark mode, and instant quote updates.

## Features

- Live quote calculation.
- Panel package pricing from 5 to 24 panels.
- Battery options for Sigenergy and Huawei.
- Automatic BER certificate inclusion.
- Ground mount pricing based on panel count.
- Optimizer pricing with quantity input.
- Multiple add-ons support.
- Dark mode toggle.
- Print-ready quote layout.
- Copy quote to clipboard.
- Mobile responsive design.
- No frameworks or external dependencies.

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript

## How to Use

1. Open the HTML file in a browser.
2. Select the number of panels.
3. Optionally enable battery, ground mount, optimizers, and add-ons.
4. Review the live quote summary.
5. Use the buttons to reset, print, or copy the quote.

## Pricing Logic

The calculator includes:

- Solar packages with fixed pricing.
- Battery pricing based on brand and size.
- Ground mount cost at €60 per panel.
- Optimizers at €75 each.
- Add-on pricing for extra installation options.
- BER certificate included automatically by default.
